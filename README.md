# Chat-App-Microservice-RabbitMQ

A microservice-based chat application leveraging **RabbitMQ** for message brokering and **TypeScript** for implementation. This project demonstrates scalable communication between services using event-driven architecture.

---

## 🚀 Features
- **Microservice Architecture** – loosely coupled services for scalability
- **RabbitMQ Integration** – reliable message queue for chat events
- **Dockerized Setup** – easy deployment with `docker-compose`
- **CI/CD Ready** – Jenkins pipeline included for automation
- **TypeScript** – strong typing and maintainable codebase

---

## 📂 Project Structure
- `.gitignore` – ignored files configuration
- `Jenkinsfile` – CI/CD pipeline definition
- `docker-compose.yml` – container orchestration
- `src/` – application source code (TypeScript services)

---

## ⚙️ Prerequisites
- Node.js (>= 18.x)
- Docker & Docker Compose
- RabbitMQ (via Docker or local installation)
